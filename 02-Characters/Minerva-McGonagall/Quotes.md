# 我的笔记

**说话动词**：极少重复，靠动词分层情绪——`said irritably / coldly / grudgingly`（不耐、冷淡、不情愿）、`faltered / repeated faintly`（震惊失态）、`hissed / whispered`（压低声音的谨慎）、`cried`（情绪爆发）。她的情绪基本由动词点出，台词本身保持克制。

**用词选择**：偏书面、略带旧式正式感。常用 `downright / in broad daylight / in the name of heaven / noble / astounding`，而不是口语化的强调词。评价人时则极简短锋利："He never had much sense."

**修辞习惯**：反问与设问频繁（"I suppose he really has gone?" / "Couldn't you do something about it?"），带训导口吻；破折号和省略号密集，用于制造迟疑、哽咽、情绪断点。句式整体工整完整，复合句多。

**称呼姿态**：坚持称 "You-Know-Who"，被纠正后才勉强改口 "Voldemort"；对邓布利多始终直呼其名但语气恭敬中带质疑；对海格的评价是"心眼不坏但毛糙"，体现她**敬重权威、但保留独立判断**的姿态。

**个人特色**：外冷内热、句正情深。平时句式严整、判断犀利；一旦动情，工整的外壳出现裂缝——开始结巴、省略、破折号成串。**克制是常态，破裂是信号**。
