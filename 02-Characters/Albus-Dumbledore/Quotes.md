# 我的笔记

## 语言特征小结（第一章）

- **说话动词**：以 _gently / calmly / firmly / finally / murmured_ 为主，几乎不带激烈情绪，越是重话越平静。
- **用词选择**：书面语（"precious little""sensible person"）夹杂生活化俏皮词（"lemon drop""earmuffs""伦敦地铁图"），雅俗混搭。
- **修辞习惯**：常用自嘲式幽默化解沉重话题；爱用省略号和 _may / perhaps_ 等模糊限定，留白多于断言。
- **称呼姿态**：对麦格用 "My dear Professor" 礼数周全，对海格直呼其名并完全信任；外壳谦和，关键决定却 _firmly_ 拍板。
- **个人特色**：坚持直呼"伏地魔"——用最平静的语气说最强硬的立场，是他最具辨识度的标记。
