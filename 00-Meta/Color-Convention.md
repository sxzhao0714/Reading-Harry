# 四色高亮约定

English Made Easy 插件使用 HTML 标签存储高亮，格式为：
`<mark class="eme-highlight eme-h-{color}" data-id="...">文字</mark>`

## 颜色含义

| 颜色  | class          | 含义                | AI 处理方式                    |
| --- | -------------- | ----------------- | -------------------------- |
| 黄色  | `eme-h-yellow` | 生词（单词或词组）         | 输出：词性 / 中文释义 / 英文简释 / 原文例句 |
| 蓝色  | `eme-h-blue`   | 精彩表达（地道说法、想模仿的句子） | 输出：含义 / 用法 / 可模仿的语境        |
| 绿色  | `eme-h-green`  | 句型 / 语法点          | 输出：语法名称 / 结构拆解 / 一个扩展例句    |
| 粉色 | `eme-h-pink` | 用户标注（人名地名等想记录的内容） | 原文 + 中文翻译 |

## 学习水平参考
- 学习者英语水平：CET-4，目标提升至 CET-6 / 雅思
- 生词解释深度按此基准
